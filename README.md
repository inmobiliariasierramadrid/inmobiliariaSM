# Inmobiliaria Sierra Madrid — web

Web estática (HTML/CSS/JS). Se despliega en Vercel como sitio estático, sin build.

- Publicada con Vercel (framework: **Other**, sin build command, output = raíz del repo).
- Formularios vía Web3Forms (clave en `data.js`, constante `SM_FORMS_KEY`).
- Gestor de propiedades en `/admin.html`.
